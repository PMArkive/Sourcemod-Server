### How to be Adm ###
1. open addons\sourcemod\configs\admins_simple.ini
2. add this line at the buttom of text, [Steamid Finder](https://steamid.xyz/)
    ```c
    "STEAM_X:X:XXXXXXXX" "99:z"
    ```
3. Save->Restart Server
